# Press Review 

## À quoi ça sert

Ce projet a pour but de faire une mini app pour faire notre press-review.

Celui-ci sera une v1 pour voir si c'est une bonne idée ou pas !

## Installation ⚙️

```shell
npm install
# Pour lancer le serveur en local
npm start 
```
